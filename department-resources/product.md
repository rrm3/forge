# Product - AI Tuesdays Resources

## Getting Started
Product managers and designers can use AI to accelerate research synthesis, generate specs, and prototype ideas faster. AI is especially powerful for the discovery and definition phases.

## By Role

### Product Managers
* Synthesize user research into actionable insights
* Try: Paste 5-10 user interview quotes and ask for themes and patterns
* Try: Draft a PRD from a problem statement and user stories
* Advanced: Use AI to generate RICE scores from a feature list with context

### UX Designers
* Generate user personas, journey maps, and wireframe descriptions
* Try: Describe a user flow and ask for usability concerns
* Advanced: Feed in heatmap data and ask for UX improvement recommendations

### Product Analysts
* Write SQL queries and analyze results in natural language
* Try: Describe the metric you want to measure and let AI write the query
* Advanced: Build a dashboard specification from business requirements

## Recommended Tools
* Claude: Research synthesis, spec writing, strategic thinking
* ChatGPT with Code Interpreter: Data analysis, charts, SQL
* Figma AI: Design prototyping and iteration
